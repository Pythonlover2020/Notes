# 概述

>C++ = C + 面向对象编程 + 泛型编程

>C语言是面向过程的，C++是面向对象的

# 初识

## Hello world

```cpp
#include <iostream> // 导入库，不需要.h后缀
// iostream 即标准输入输出流

int main()
{
    std::cout << "Hello world\n"; // 这里留到后面讲

    return 0;
}
```

Hello world
